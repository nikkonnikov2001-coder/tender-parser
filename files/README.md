# 🏛 Тендерный AI-бот

Telegram-бот для автоматического мониторинга и AI-анализа государственных закупок на [zakupki.gov.ru](https://zakupki.gov.ru).

## Возможности

- **🔍 Поиск тендеров** — парсинг ЕИС по ключевым словам, цене, регионам и законам
- **📄 Скачивание ТЗ** — автоматическая загрузка документации из карточки тендера
- **🤖 AI-анализ** — нейросеть (Ollama) читает ТЗ и выделяет предмет контракта, сроки, штрафы
- **📊 Excel-отчёт** — аналитическая таблица с результатами, доставляемая прямо в чат
- **⏰ Мониторинг** — автоматическая проверка новых тендеров по расписанию
- **⚙️ Гибкие настройки** — всё управляется через интерфейс бота

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
playwright install chromium
```

### 2. Настройка .env

```bash
cp .env.example .env
# Заполните TELEGRAM_BOT_TOKEN и TELEGRAM_CHAT_ID
```

### 3. Запуск Ollama (для AI-анализа)

```bash
ollama pull qwen2.5:7b
ollama serve
```

### 4. Запуск бота

```bash
python bot.py
```

## Структура проекта

```
├── bot.py            # Главный файл бота (меню, обработчики, мониторинг)
├── bot_config.py     # Управление настройками (JSON-хранилище)
├── bot_search.py     # Конвейер поиска с прогрессом в Telegram
├── parser.py         # Парсер страницы результатов zakupki.gov.ru
├── downloader.py     # Скачивание документов из карточки тендера
├── reader.py         # Чтение .docx (абзацы + таблицы)
├── analyzer.py       # AI-анализ ТЗ через Ollama
├── notifier.py       # Отправка отчёта (legacy)
├── browser_ctx.py    # Настройки Playwright (UA, viewport)
├── tz_docs.py        # Фильтр: какие файлы считать ТЗ
├── load_env.py       # Загрузка .env
├── requirements.txt  # Зависимости
└── .env.example      # Шаблон переменных окружения
```

## Технологии

- **Python 3.11+**
- **aiogram 3** — Telegram Bot API
- **Playwright** — headless-браузер для парсинга ЕИС
- **BeautifulSoup** — парсинг HTML
- **Ollama** — локальная LLM для анализа документов
- **pandas + openpyxl** — формирование Excel
- **APScheduler** — планировщик для мониторинга
